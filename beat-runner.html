Overview

Beat Runner is a browser based 3D rhythm runner where obstacle pacing is synced to the tempo of a selected song. I built it as a complete prototype: game feel, progression, a small economy loop, and a playable demo.

Goal

Validate if a cosmetic-grind loop works in a skill-based rhythm game.

My Role

As the sole designer and developer, I handled:

  • Core runner loop and tuning
  • Stage Mode: 15-level difficulty curve and star ratings
  • Free Run mode with boosters (Magnet, Shield, Speed Boost)
  • Orbs earn rate and cosmetic pricing
  • UI and basic persistence for progression

What I Built

Stage Mode

15 levels with increasing note density to teach mechanics.

Economy

Soft currency (Orbs) earned via play, spent on cosmetic trails.

Progression

Star-gating system to enforce mastery before advancement.

Iteration & Insight

Players churned early when difficulty spiked too fast. I smoothed the note density curve in levels 3-5, increasing early retention.

Result

Delivered a performant 3D web prototype proving the core loop is fun and the economy supports long-term engagement.

What's Next

  • Tighten mobile controls and performance profiling
  • Add lightweight challenges (weekly goals)
  • Expand cosmetics (trails, particles)
  • Consider async ghost runs if scope stays small
