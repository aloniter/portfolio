Inspector Pro

iOS building inspection reporting app prototype: capture photos, mark findings, and export a clean Word report fast.

Used in production by a family-run inspection company.

Role Product + Builder (solo)
Platform iPhone (iOS)
Timeline Shipped (Production use)
Type Prototype / MVP

The Problem

Building inspections generate lots of photos and notes, but turning them into a structured report is slow and messy. The goal was to reduce admin time and standardize reports so inspections can be delivered quickly and consistently.

What I Built (MVP Scope)

  • Create a project per inspection (site / date / notes)
  • Capture or upload photos directly into the project
  • Annotate photos (draw, highlight, add notes)
  • Auto-generate a numbered findings list from marked items
  • Export a structured Word report (photos + numbered findings table)

Product Thinking (Why these choices)

  • "Photos first" flow: inspectors already work visually, so the app matches real behavior
  • Findings table is generated automatically to prevent missed items and save time
  • Word export is the final deliverable format most clients/teams actually need

What I Learned

  • The real pain isn't taking photos — it's organizing them into a report
  • A fast annotation experience matters more than fancy UI
  • Report structure needs to be consistent (numbering + titles + location fields)

Next Steps

  • Templates (different report formats per client/company)
  • User accounts so each inspector has their own projects
  • Sharing: allow a company workspace with shared projects
  • Faster batch actions (add location/tag to multiple photos)
  • Export improvements (cover page, signatures, branding)

Tech Notes (keep simple)

  • iPhone app prototype focused on workflow + export output
  • Data model built around Projects → Photos → Findings
  • Export pipeline to produce a clean, client-ready Word document

Inspector Pro is my attempt to turn a repetitive, manual inspection workflow into a fast, reliable "capture → mark → export" loop.
